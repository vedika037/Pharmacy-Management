# Pharmacy Management System

Overview
The Pharmacy Management System is a Flask-based web application designed to help manage medicines, customers, and delivery records efficiently.  
It provides an easy-to-use interface for pharmacy staff to handle day-to-day operations like adding medicines, managing customers, tracking deliveries, and maintaining audit logs

Features
- Add, update, and delete medicine details  
- Manage customer information and purchases  
- Maintain delivery person details and logs  
- Record all customer actions in an audit trail  
- Simple and intuitive web interface built with HTML, CSS, and JavaScript  
- Connected with MySQL database for persistent storage  

Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask Framework)  
Database: MySQL  
Version Control: Git & GitHub
